# Advanced Excel Project: JOB Market Intelligence Analysis

# Objective
Analyze the Job Market dataset to generate actionable HR insights for hiring strategy, salary benchmarking, demand forecasting, remote work trends, and AI job market growth.

# Tech Stack

- Microsoft Excel (Advanced)

- Pivot Tables & Pivot Charts

- Power Query (if used)

- Formulas (XLOOKUP, FILTER, UNIQUE, SORT, IF, SUBTOTAL, etc.)

- Data Validation & Conditional Formatting

- What-If Analysis


# Steps Included

>- Data Cleaning & Preparation

>- KPI Calculations

>- Advanced Formula Implementation

>- Dynamic Filtering & Sorting

>- Dashboard Creation

>- Scenario Analysis (Salary Simulation)


# Visualizations Included
- Pivot Charts (Salary vs Experience, Industry Distribution)
- Pie Chart (Job Distribution by Industry)
- KPI Cards (Total Jobs, Avg Salary, Max Salary)
- Dynamic Dashboard with Slicers

# Analysis Included
- Remote vs On-site Salary Comparison
- High Demand Job Identification
- AI Job Market Growth Trends
- Salary Benchmarking by Industry & Role
- Top Paying Jobs & High ROI Roles
- Demand & Growth-Based Job Categorization

# Key Insights
- High-demand roles (Demand > 90) offer significantly higher salaries
- Remote jobs tend to have competitive or higher salary packages
- AI & Technology roles dominate top salary brackets
- Certain country + role combinations provide highest ROI
- Finance & Tech industries lead in salary payouts


# Features
- Dynamic dashboards (auto-update with new data)
- Dropdown-based Job Search Tool
- Job Comparison System (side-by-side salary)
- Top 10 Salary Highlighting (auto-updating)
- Top 3 Salaries by Country (dynamic)
- Fully automated using formulas & slicers

# Conclusion

This project helps HR teams make data-driven decisions by identifying high-paying, high-demand roles and optimizing hiring strategies through interactive Excel dashboards.
